---
title: "Get Registrations"
description: "The Get Registrations API retrieves a paginated list of registrations."
api: GET https://api.trykintsugi.com/v1/registrations
contentType: application/json
---

This API helps in tracking and managing registrations efficiently across multiple jurisdictions.

## Query Params

<ParamField query="status__in" type="string" >
default `REGISTERED`, `PROCESSING`, `UNREGISTERED`, `DEREGISTERING`, `DEREGISTERED`, `VALIDATING`, `AWAITING_CLARIFICATION`

  Filter registrations by status. Multiple statuses can be passed, separated by commas.
</ParamField>

<ParamField query="country_code" type="string">
  Filter registrations by country code in ISO 3166-1 alpha-2 format (e.g., US, CA).
</ParamField>

<ParamField query="state_code" type="string">
  Filter registrations by state code.
</ParamField>

<ParamField query="filing_frequency__in" type="string">
  Filter registrations by filing frequency. Multiple filing frequencies can be passed, separated by commas.
</ParamField>

<ParamField query="page" type="integer" default="1" required>
  Page number. Must be â‰¥ 1.
</ParamField>

<ParamField query="size" type="integer" default="50">
  Page size. Must be between 1 and 100.
</ParamField>

## Headers

<ParamField header="x-organization-id" type="string" required>
  The unique identifier for the organization making the request
</ParamField>
