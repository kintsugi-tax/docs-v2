---
title: "Get Registration By Id"
description: "The Get Registration By ID API retrieves a single registration record
based on its unique identifier."
api: GET https://api.trykintsugi.com/v1/registrations/{registration_id}
contentType: application/json
---

## Path Params

<ParamField path="registration_id" type="string" required>
  The unique identifier of the registration to retrieve.
</ParamField>

## Headers

<ParamField header="x-organization-id" type="string" required>
  The unique identifier for the organization making the request
</ParamField>
