## Overview

Kintsugi API implements rate limiting to ensure fair usage and system stability.

## Rate Limits

- **Standard endpoints**: 1000 requests per minute
- **Tax calculation**: 500 requests per minute
- **Bulk operations**: 100 requests per minute
